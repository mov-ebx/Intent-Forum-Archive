      Note: I tested these methods using a VM because I wanted to use a VPN and I was unable to on my windows pc because I don't have admin. That means that is is by no means guaranteed to work as I may have screwed something up.
      Step 1: Download Java. This is probably the easiest part and can be done by following this guide: https://www.guru99.com/how-to-install-java-on-ubuntu.html
      Step 2: Download the Intent Launcher at intent.store/launcher
      Step 3: Drag the intent launcher.jar to your desktop
      Step 4: Right click on the .jar select properties and once the properties tab opens up click permissions and check the box next to execute that says "Allow executing file as program" Note: If you see a popup telling you this file could be malicious blah blah blah, ignore it and follow the steps I just told you to get rid of it.
      Step 5: Install a VPN. I used proton because I don't have any premium VPNs I could use. If you have a different VPN look on their website for info on using it on Linux. Proton has a very nice tutorial for using their VPN so I won't rehash that here.
      Step 6: Run the intent launcher, load up your client, turn on your VPN and have fun!
