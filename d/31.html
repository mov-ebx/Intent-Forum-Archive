    If you are having trouble with the intent launcher even after 5.3, or you just prefer to use the original launcher.

    1. Place the ZeroDay folder containing the ZeroDay.jar and .json into .minecraft/versions
    2. Create a profile in the launcher and select release ZeroDay
    3. Open the advanced settings at the bottom and press browse
    4. Find a javaw.exe in Program Files/Java/<your version>/bin/ and save the profile
    5. When you launch the client, it should not crash.

    These instructions apply to Mac and Linux as well, you however need to find the respective java executable for your operating system.
