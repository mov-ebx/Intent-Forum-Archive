    Here's a list of ways to fix current (usually rare) errors:

    • Zeroday crashes when I launch it (ERROR CODE 1): Inside the launcher, click the settings icon and slide down the slider a little bit (that's all you have to do). This decreases the allocated RAM, which can lead to FPS lag spikes. If you want to allocate more than 2 GBs of RAM, please download 64-bit Java.
    • Whitelist error: double-check that your system clock is set correctly, make sure to set it to automatic
    • Game crashes when you join a server (Fixed in b20): goto the baritone folder located at C:\Users\YOU\intentlauncher\baritone and make a text file called settings.txt and keep it empty. If you already have this folder, delete it and make an empty one.
    • EXCEPTION_ACCESS_VIOLATION (Should be fixed): Just press launch again.
    • Nothing happens when I run the launcher: Most likely due to outdated graphics drivers, https://www.drivereasy.com/knowledge/error-org-lwjgl-lwjglexception-pixel-format-not-accelerated/

    If any new errors start happening or Nef patches these ones, I will keep this post up to date.

    IF YOU HAVE A DIFFERENT PROBLEM/ERROR, CONTACT ME ON SLACK OR MAKE A NEW POST ON THIS FORUM

          And just because, I'll put the rarity of each in this reply:

          • Whitelist error: Common, you wouldn't believe how many people don't have their clocks set correctly.
          • Launcher doesn't run: VERY rare, only happened to one or two people in the history of ZeroDay, and it's lwjgl's (A professionally coded library that Minecraft uses) fault.
          • ZeroDay crash: Common, lots of people don't have 64-bit Java
          • EXCEPTION_ACCESS_VIOLATION: Happens to everyone, all you need to do is press launch again. Soon to be fixed by Nef, most likely will be fixed by b20
          • Game crashes when you join a server: Uncommon, I'm not sure why it happens but it sure it easy to fix. Fixed in b20

            • Zeroday crashes when I launch it (ERROR CODE 1) (Fixed in b20): Uninstall all versions of java you currently have downloaded by

               1. Click the Windows button
 2. Select Settings
 3. Select Apps
 4. Search "Java"
 5. Select Java and then click its Uninstall button.
 6. Respond to the prompts to complete the uninstall
 7. Repeat step 5 for any other installations of Java
               Then, download and install this version: https://intent.store/download-jdk
