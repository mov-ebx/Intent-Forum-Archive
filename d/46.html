      For all my Linux boys out there this ones for you!
      Some of you may be having issues with setting up Zero day on your distro and Im here to help.

      Tutorial (Debian though most things will be the same):
      1- Install JAVA (may seems simple but a lot of newbies don't know how)
      Just download if from the website and follow the tutorial, don't add any extra repositories.

      2- Installing MultiMC
      Download from MultiMC.org, run and click install on the deb file.
      On first startup it asks what version of java you want to use, and use the one with a start next to it.
      Login in the top right.
      Once your logged in you can either click create instance in the top left, or right click and select create instance.
      In this example I'm using Zero day so in version select 1.8.8 and hit ok.

      3- Installing Zero day
      Download client files from intent.store and extract them.
      Now you want to right click on the instance you created and hit Edit Instance.
      On the right click Replace Minecraft.jar and select the jar from your client (ZeroDay.jar)
      Hit launch, and your good to go!
