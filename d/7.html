        It doesn’t take that long, if it’s successful. What probably happened is an unhandled error during the install process, maybe a dip in connection or a system permission violation. You could delete the C:/Users/<you>/intentlauncher folder and try again, that has some success. You could also try running it in command prompt like

        java -jar <drag in launcher jar>

        and see if any visible errors occur during install and post those. In the next version I’m going to handle more edge cases that cause things like this, and make more errors visible without command prompt.
